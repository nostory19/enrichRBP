The tutorial for web server of enrichRBP:

Imputation:

To use enrichRBP to predict and analyze RBP binding sites on linear and circular RNAs, you should:

1.Upload the file or input the sequence, the length of the sequence must meet the fixed length of 101 and ensure that the file format is the same as the provided sample.

2.Choose the imputation methods you would like to perform.

3.Choose which prediction model you want to use. For the circRNAs binding site prediction task, we provided 37 prediction models per imputation and 31 prediction models per imputation for the linear RNA binding site prediction task.

4.Enter your email address to get a unique ID to view your progress.

5.Click “Submit” to submit your task.

6.Return to the home page and enter your unique ID to see the results of the task.
